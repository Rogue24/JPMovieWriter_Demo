# JPMovieWriter_Demo

**JPMovieWriter**是基于`GPUImageMovieWriter`改造的录制器，可暂停、续录、定时、且带有滤镜美颜，另附GIF制作。

![](https://github.com/Rogue24/JPCover/raw/master/JPMovieWriter_Demo/JPMovieWriter_0.gif)

    Feature：
        ✅ 可暂停、定时、续录；
        ✅ 多种滤镜选择；
        ✅ 基础美颜功能；
        ✅ 支持相册视频的导入和拼接；
        ✅ GIF制作；
